# ⚔️ AION 2 幣值換算最佳化工具 (AION 2 Currency Exchange Optimizer)

專為《AION 2》玩家打造的單頁面靜態網頁工具，用於精算新台幣 (NTD) 轉換為遊戲幣 (基納) 的最高效益途徑。

本工具不僅對比「官方/代儲寶石」與「第三方幣商」的真實匯率，更深度結合了**真實遊戲市場的庫存限制**與**多重手續費耗損**，幫助玩家將每一塊錢的預算榨出最大價值的遊戲幣。

目前介面已朝向與 craft 工具一致的視覺語言整理，採用深色背景、卡片區塊、發光邊框與響應式排版，讓兩個工具在同一個專案內看起來像同一套系列介面。

## ✨ 核心功能 (Features)

* 🛒 **賣場庫存限制與混搭策略**：獨家引入「賣場剩餘箱數」參數。當高面額寶箱數量不足時，系統會自動啟動「混搭購買」模式，將剩餘的台幣餘額轉投資於低面額寶箱，實現預算零浪費。
* 📊 **四大方案綜合評比**：一鍵運算並比較「混搭購買」、「單買 1000 萬箱」、「單買 200 萬箱」與「第三方幣商 (交易所拉裝)」四種方案的最終實拿基納，並自動高亮標示 [總量最多] 的最佳解。
* 🧮 **隱藏稅金精準扣除**：完整內建交易所雙重手續費 (上架 2% + 交易完成 10%)，並支援自訂第三方交易平台 (如 8591) 的額外抽成比例，讓玩家看見被系統吃掉的真實成本。
* 💾 **無縫記憶體驗**：輸入過的各項浮動數值 (如代儲物價、幣商匯率、賣場庫存) 會透過瀏覽器的 Local Storage 自動儲存，下次開啟網頁無需重新設定。
* 📱 **響應式網頁設計 (RWD)**：介面簡潔直觀，完美支援電腦與智慧型手機操作，方便邊玩遊戲邊用手機試算。

## 🛠️ 核心計算邏輯 (How it Works)

1. **寶石成本單價**：`花費台幣 ÷ 獲得寶石數量 = 單顆寶石台幣成本`
2. **寶箱收益 (含庫存判定)**：根據「總預算可買箱數」與「賣場可買箱數」取最小值進行計算，並精算剩餘台幣。
3. **混搭購買策略**：優先以最高預算買滿 1000 萬箱 ➔ 計算台幣餘額 ➔ 將剩餘台幣投入 200 萬箱 ➔ 加總雙邊基納收益。
4. **幣商收購實拿**：`總預算 × 幣商匯率 × (1 - (上架手續費% + 交易手續費% + 平台額外抽成%))`
