# AION2 裝備製作成本計算器 (Crafting Cost Calculator)

這是一個專為《AION2 (永恆紀元2)》打造的單頁式 (SPA) 裝備材料與製作成本試算工具。
本專案為了解決靜態伺服器 (如 GitHub Pages) 針對中文路徑大小寫或編碼常發生的 `404 找不到檔案` (破圖) 問題，經歷了多次架構重構，目前採用**「模組化開發、Python 強制編譯、單一檔案輸出」**的架構。

## ✨ 核心特色
1. **多件合併計算：** 可以透過「系列」挑選多把武器、臂甲加入清單，系統會自動在下方幫您「合併加總」所有需要的素材。
2. **顏色等級與文字優化：** 所有道具名稱預設剔除了 `(刻印)` 後綴，且英雄級 (Epic) 使用專屬橘色 `#ffa500`，獨特級 (Unique) 使用專屬黃色 `#ffd700`。
3. **系列化選單設計：** 捨棄了過去冗長的單件列表，改以系列 (`夔龍王系列`、`應龍王系列`...等) 為單位，點擊後再以「下拉選單」選擇真正的職業武器，讓介面乾淨好讀。
4. **自動英文化防破圖：** 系統編譯時，會自動將各種中文路徑 (`應龍王巨劍.png`) 翻譯成英文 (`YinglongKingGreatsword.png`)，一勞永逸地解決破圖問題。

```text
nca2-craft-tool/
├── css/
│   └── style.css            # 原始 CSS 樣式檔 (請編輯此檔)
├── js/
│   ├── items.js             # 原始物品辭典 (圖片對應、品階、武器類型)

├── app_series_cart.js       # 購物車邏輯、系列轉換與 UI 渲染程式碼
├── build_all.py             # 專案編譯與轉換工具
├── README.md                # 本說明文件
└── index.html               # 【請勿手動編輯】由 build_all.py 自動生成的最終產物
```

## 🛠️ 開發與編譯流程 (如何更新資料)

**請注意：絕對不要直接去修改 `nca2-craft-tool/index.html` 裡面的邏輯或資料！**
因為它是被腳本「生出來」的壓縮檔。

### 新增配方或修改資料 (正常流程)：
1. 改數值或掛載圖片：請編輯 `js/items.js` (定義道具有甚麼圖、屬性) 以及 `js/recipes.js` (定義要吃甚麼材料)。
   - 若要新增圖片，請將圖片放入 `images/` 中，如果檔名是中文也無妨。
2. 開啟終端機 (Terminal / PowerShell)，進入 `nca2-craft-tool/` 目錄。
3. 執行編譯指令：
   ```bash
   python build_all.py
   ```
4. **編譯器 (`build_all.py`) 會自動執行以下動作：**
   - 掃描 `items.js`、`recipes.js` 與 `style.css`。
   - 將任何有 `(刻印)` 的字眼移除。
   - 動態將 HTML / JS 中參照到的中文圖片名稱翻譯成英文，並且**同時去更改硬碟上的圖片檔名**。
   - 解決所有文字編碼 (Big5 / UTF-8) 衝突，最終一鍋煮成完美的 `nca2-craft-tool/index.html`。
5. 使用 Git 提交變更 (Commit & Push)：
   ```bash
   git add .
   git commit -m "Update recipes"
   git push
   ```
6. 等待 GitHub Pages 佈署完成 (約 30~60 秒)，使用者只要按下 `Ctrl + F5` 即可看到最新資料！
